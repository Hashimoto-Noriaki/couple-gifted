# サービス概要

## サービス名

CoupleGifted（カップルギフテッド）

## コンセプト

「パートナーが喜ぶ」を軸にした、カップル・夫婦向けのライフプラットフォーム。
交際中から結婚後まで長く使えるサービスを目指す。

## 解決する課題

- デートスポットを「相手目線・カップル目線」で探せるサービスがない
- 食べログ・Googleマップの口コミはカップル視点ではない
- 将来的にはプレゼント・コスメ・体験ギフトにも展開

## 競合との差別化

- 食べログ／Googleマップ → カップル特化の視点がない
- LIPS／@cosme → 女性向けで男性・カップル視点がない
- Pairy／Pairto → スポット・コスメ情報がない
