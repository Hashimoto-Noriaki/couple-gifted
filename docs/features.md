# 機能一覧と展開計画

## Phase 1（MVP・現在開発中）

デートスポット検索に特化。

### 開発順序

1. ユーザー認証（会員登録・ログイン）
2. Spot・Tagモデル設計
3. スポット一覧・詳細API（シーン別・エリア検索）
4. 口コミ投稿（SpotReview）
5. いいね（Like）

### 機能一覧

- シーン別検索（記念日・初デート・雨の日・夫婦おでかけなど）
- エリア検索
- Google Places API連携でスポットデータ取得
- カップル・夫婦目線の口コミ投稿
- 評価（1〜5）＋本文
- いいね機能
- ユーザー認証（会員登録・ログイン）
- 投稿者属性表示（交際中／婚約中／既婚）

## Phase 2

- コスメ・プレゼント検索（楽天API連携）
- 男性が彼女向けコスメを探す
- 女性が彼氏向けメンズコスメを探す
- プレゼントカテゴリ追加（ファッション・体験ギフト等）

## Phase 3

- AIによるプレゼント・デートプラン診断
- 記念日リマインド通知
- ブランド・飲食店の公式アカウント機能
- 夫婦向け暮らし・インテリア情報
- 広告・タイアップ機能
