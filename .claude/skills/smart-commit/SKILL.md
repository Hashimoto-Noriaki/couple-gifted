---
description: ステージ済みの差分からコミットメッセージを生成する
---

`git diff --cached` でステージ済みの差分を確認し、コミットメッセージを提案する。

## プレフィックス規則

| prefix | 用途 |
|--------|------|
| feat   | 新機能追加 |
| fix    | バグ修正 |
| test   | テストのみの追加・修正 |
| refactor | リファクタリング |
| docs   | ドキュメント更新 |
| chore  | 設定・ツール変更 |

## 注意

- 実装ファイルとテストファイルが混在している場合は `feat:` + `test:` に分割して提案する
- メッセージは 1 行で完結させる（本文不要）
- ステージされていないファイルがある場合は一覧を表示して確認を促す
